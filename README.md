🎬 Movie Rental System - Web Application

This project is a Movie Rental Web Application developed using PHP and MySQL.
It allows users to browse movies, add them to a cart, and rent them online.

📝 Objectives
To develop a dynamic web application using PHP, HTML, and CSS
To implement a user authentication system (Login/Register)
To practice CRUD operations (Create, Read, Update, Delete)
To connect and manage data using a MySQL database
🎮 How It Works

The system allows users to rent movies through a simple flow:

Browse available movies on the homepage
Add selected movies to the cart
Login/Register to the system
Confirm and complete the rental
